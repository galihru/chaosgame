# Chaos Game

## 一、项目简介

本项目基于**混沌游戏（Chaos Game）**算法生成分形图形，当前实现为经典的**谢尔宾斯基三角形（Sierpiński Triangle）**。

通过随机迭代点在顶点之间的线性收缩映射，本系统在二维平面上生成具有自相似结构的分形几何图案。

从数学本质上看，该系统属于**迭代函数系统（Iterated Function System, IFS）**的一种实现形式。

---

## 二、数学模型与公式

设三角形三个顶点为：

[
A(x_1, y_1), \quad B(x_2, y_2), \quad C(x_3, y_3)
]

任取初始点：

[
P_0(x_0, y_0)
]

在第 ( n ) 次迭代时：

1. 随机选择一个顶点 ( V \in {A, B, C} )
2. 按比例 ( r ) 向该顶点移动

更新公式为：

[
P_{n+1} = (1 - r)P_n + rV
]

当：

[
r = \frac{1}{2}
]

时，系统收敛到经典的谢尔宾斯基三角形吸引子。

---

## 三、几何与分形性质

### 1️⃣ 自相似性（Self-Similarity）

每个大三角形由三个缩放比例为：

[
\frac{1}{2}
]

的小三角形组成。

### 2️⃣ 分形维数（Fractal Dimension）

谢尔宾斯基三角形的豪斯多夫维数为：

[
D = \frac{\log 3}{\log 2} \approx 1.585
]

说明该结构：

* 维数大于1（非线）
* 维数小于2（非平面填充）

属于非整数维分形结构。

## 五、为什么会出现“黑色空洞”？

黑色区域并非未绘制，而是：

根据迭代函数系统的收缩映射原理，某些区域**不属于吸引子（Attractor）集合**。

数学上：

该分形为吸引子集合：

[
S = \bigcup_{i=1}^{3} f_i(S)
]

其中：

[
f_i(x) = \frac{1}{2}(x + V_i)
]

黑色区域表示不满足递归自相似条件的点集。

---

## 六、为什么不是“无限多不同形状”？

在：

* 顶点固定
* 比例固定 ( r = 1/2 )
* 无附加约束

条件下，

系统严格收敛到唯一吸引子 —— 谢尔宾斯基三角形。

若想生成不同分形，需要：

* 改变 ratio
* 增加顶点数
* 添加选择限制规则
* 或引入 AI 动态策略

---

## 七、AI 扩展模式（理论说明）

若启用 AI 模式：

顶点选择概率可由策略函数控制：

[
\pi(V_i) = \frac{e^{Q_i / T}}{\sum_j e^{Q_j / T}}
]

其中：

* ( Q_i )：顶点价值函数
* ( T )：温度参数

此时系统可能偏向某些区域，产生非对称分形结构。
